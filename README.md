# Week 5 Day 3 : Allocinoche

Some fun with DB using Active Record !

## Run Locally

Clone the project

```bash
  git clone https://github.com/WinnyDrancourt/week_5_day_1.git
```

Go to the project directory

Install dependencies

```bash
  bundle Install
```

Then

```bash
rails c
```
